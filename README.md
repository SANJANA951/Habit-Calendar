👉 [Click here to open app](https://SANJANA951.github.io/Habit-Calendar/)

## ✨ Features

- 📆 Full monthly calendar with proper alignment  
- 🔄 Previous/Next navigation + Today shortcut  
- 🎯 Habit tracking with colored status dots  
- 📊 Daily progress bar per date  
- 🟢 Green (all done) / 🟣 Purple (partial) visual feedback  
- 📝 Add, edit, and delete events/notes  
- 🎨 Color-coded event pills on calendar  
- 💾 Data saved using localStorage  
- ⚡ Smooth UI animations
